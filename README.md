# E-commerce
Outil: PHP (MVC)

Site d'Internet pour les agences touristiques.

-Plateform administrateur:
+Gérer lés tours (supprimer, ajouter, modifier les tours)
+Gérer les comptes des clients

-Plateform client:
+Créer son compte
+Ajouter les tours
+Supprimer les tours
+Payer les commandes


Outils : HTML, CSS, Javascript, Boostrap, PHP(MVC),...
